# 프로모션 운영·검증 시스템

## 프로젝트 개요

전북 AI디지털배움터 강사용 통합 프로모션 포털

강사는 하나의 사이트에서 진행 중인 프로모션을 확인하고 참여할 수 있으며,
관리자는 프로모션 운영, 검수, 결과 집계 및 정산 관리를 수행한다.

---

## 강사용 사이트

### 메인화면

* 진행 중인 프로모션 카드 표시
* 최대 5개 프로모션 노출
* 이미지 카드 클릭 시 참여 페이지 이동

### 기능

* 프로모션 참여 등록
* 내 제출 현황 조회
* 내 결과 조회
* 반려 사유 확인
* 예상 지급액 확인

---

## 관리자 사이트

### 대시보드

* 전체 제출 건수
* 검수 대기
* 인정
* 반려
* 중복 의심

### 프로모션 관리

* 프로모션 생성
* 프로모션 수정
* 프로모션 종료
* 운영기간 설정
* 지급기준 설정

### 검수

* 승인
* 반려
* 반려 사유 입력

### 결과

* 프로모션별 결과 조회
* 강사별 결과 조회
* CSV 다운로드
